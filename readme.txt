== Description ==

VP Slider is simple and easy to use. This slider you can use any page and file. 
Demo: http://plugin.onjumpstarter.io/?page_id=23


<strong>Shortcode</strong>

<strong>Simple Slider</strong>

<pre>[VP_Slider]</pre>

<strong>slide per page</strong>
if you want to use more VP Slider in 1 page, you must use <pre>id</pre>
<pre>[VP_Slider id="1" ppp="3"]</pre>

<strong>slider category support </strong>
<pre>[VP_Slider id="2" cat="new"]</pre>

<strong>slider stop autoPlay</strong>
<pre>[VP_Slider id="3" autoPlay="false"]</pre>

<strong>slider stop stopOnHover</strong>
<pre>[VP_Slider id="4" stopOnHover="false"]</pre>

<strong>slider navigation</strong>
<pre>[VP_Slider id="5" navigation="true"]</pre>

<strong>slider pagination</strong>
<pre>[VP_Slider id="6" pagination="true"]</pre>

<strong>slider responsive false</strong>
<pre>[VP_Slider id="7" responsive="false"]</pre>

<strong>slider title font size</strong>
<pre>[VP_Slider id="8" title="15"]</pre>

<strong>slider speed(milliseconds)</strong>
<pre>[VP_Slider id="9" sec="100"]</pre>
